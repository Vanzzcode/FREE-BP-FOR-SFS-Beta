Cara Setup Supabase

1
Daftar gratis di supabase.com → buat project baru
2
Buka Storage → klik New bucket → beri nama (misal: media) → set ke Public
3
Buka Settings → API → copy Project URL dan anon public key
4
Di Storage → klik bucket → Policies → tambah policy: allow all operations untuk role anon
