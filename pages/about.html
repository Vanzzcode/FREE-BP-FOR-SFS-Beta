    ℹ️ About This Website

    Website koleksi blueprint gratis untuk komunitas Space Flight Simulator. Semua blueprint dikurasi dari berbagai kontributor komunitas.

    🚀

    Versi Website

    v2.3.0

    Major update dengan sistem backend Supabase, upload komunitas, panel admin/moderator, komentar, like, dan banyak lagi.

    📅

    Terakhir Diupdate

    Mei 2026

    Update terbesar sejak rilis pertama. Migrasi dari localStorage ke Supabase backend.

    🎮

    Game

    SFS

    Space Flight Simulator — game simulasi roket mobile yang populer di seluruh dunia.

    🌐

    Platform

    Web App

    HTML, CSS, JavaScript + Supabase backend. Ringan dan cepat di semua perangkat.

    💾

    Backend

    Supabase

    Statistik, blueprint, komentar, dan user tersimpan di database Supabase yang aman dan real-time.

    👥

    Komunitas

    Terbuka

    Siapapun bisa register dan upload blueprint mereka sendiri ke website ini.

    📋 Changelog

    v2.3.0
    Major Backend Upgrade — Supabase Migrasi penuh ke Supabase. Fitur baru: upload blueprint oleh komunitas, sistem login/register (username+password), panel admin dengan verifikasi password, panel moderator, panel owner dengan manajemen role, komentar, like, search & filter & sort, pagination, leaderboard, halaman detail blueprint, profil user, sistem laporan, peringatan (3x = auto ban), ban/unban user, pengumuman admin, log aktivitas, Supabase Storage untuk gambar baru (hybrid dengan base64 lama), centang biru (verified), tag Kreator.
    v1.3.0
    Penghapusan Halaman Apollo Halaman Apollo digabung ke NASA Rocket. Perubahan nama halaman Artemis menjadi NASA Rocket.
    v1.2.0
    Statistik Sidebar & About Page Statistik pengunjung, total BP, dan total download dipindahkan ke sidebar. Tambah halaman About. Tombol BP yang belum ada linknya otomatis disabled. Tambah label "By ???".
    v1.1.0
    Lightbox & Preview Gambar Klik foto di card untuk melihat preview full screen dengan animasi smooth.
    v1.0.0
    Rilis Pertama Website dasar dengan 5 kategori, counter pengunjung, counter download, animasi bintang, dan hamburger menu.
